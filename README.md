## Amiga Binaries

AGA (PAL) machine required.

For safety, please boot without `Startup-Sequence` and then run the program, as the OS won't be spared.

| Program | Description | Min. Fast RAM |
| --- | --- | --- |
| [`kbug.exe`](https://github.com/burkemortlock/bin/raw/refs/heads/main/kbug.exe) | plays a short musical loop using hardware amplitude modulation | 0 |
| [`pnauts.exe`](https://github.com/burkemortlock/bin/raw/refs/heads/main/pnauts.exe) | like `kbug.exe` but plays a complete tune | 32 MB |
| [`pnauts2.exe`](https://github.com/burkemortlock/bin/raw/refs/heads/main/pnauts2.exe) | shorter version of `pnauts.exe` | 12 MB |
